# dotfiles

This is my dotfiles repo. It uses [chezmoi](https://github.com/twpayne/chezmoi) to apply the config on my machines. It is designed to work across Mac, Debian, Ubuntu and Fedora.

```sh
# To install on a new machine run
sh -c "$(curl -fsLS get.chezmoi.io/lb)" -- init --apply "BobcatProgrammer"
```

## Project Goals

* Make it easy to set up a new machine
* Make it easy to keep multiple machines in sync
* Unified experience across Mac, Debian, Ubuntu and Fedora
